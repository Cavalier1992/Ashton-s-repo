# C3 Writing Assessment

Welcome to C3's writing assessment! This assignment involves editing a topic about `git rebase`.

## The ask

This assignment evaluates your ability to edit and improve technical content. Your task is to review the provided content in `assignment.md` and make the necessary edits to enhance clarity, accuracy, and overall quality.

1. Create a new repository and upload `assignment.md` to your repo. DO NOT FORK THIS REPO.
2. Create a new branch.
3. Make changes directly to `assignment.md` on that branch. Commit and push your changes to your repo.
4. Open a PR. In the PR description, explain your changes or suggest additional improvements.
5. Email us the link to your repo when you're done!

Good luck!

## Things to look out for

* Headings and topic layout should be in a good format (for example, an `.md` file should start with an H1)
* Headings should be in sentence case
* Lists should be formatted correctly
* Use present tense when possible

Refer to the [Microsoft Writing Style Guide](https://learn.microsoft.com/en-us/style-guide/welcome/) for any style guide considerations.
